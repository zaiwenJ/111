# 差距分析页面重构 (Apple-Inspired Design) Spec

## Why
当前差距分析页面的功能布局已经实现，但视觉设计较为基础。用户希望将其升级为现代极简的浅色系类苹果（Apple-inspired）风格，提升页面的高级感和质感，打造符合 SaaS 标准和高端用户体验的前端界面。

## What Changes
- 全局排版升级：引入 Apple 风格的排版规范（系统级无衬线字体，字重对比，紧凑字距）。
- 质感与色彩优化：使用极简浅灰背景（如 `#f5f5f7` 或 `#fbfbfd`）、纯白卡片、类 Apple Blue 作为主色调。
- 玻璃拟态与阴影：强化 Sticky Header 和底部 Smart Dock 的毛玻璃效果（`backdrop-blur`），重构阴影层次，使其更加柔和扩散。
- 形状与圆角：采用更大的平滑圆角（`rounded-2xl`, `rounded-3xl`），使卡片和容器边缘更加圆润自然。
- 微交互提升：优化拖拽时的视觉反馈、Hover 态的平滑过渡，以及底部智能控制台展开的贝塞尔曲线动画。

## Impact
- Affected specs: 前端视觉样式、拖拽交互视觉反馈、底部状态栏与控制台动画。
- Affected code: `差距分析预览页面.html`

## MODIFIED Requirements
### Requirement: 差距分析页面视觉升级
系统 SHALL 提供类苹果（Apple-inspired）风格的差距分析界面：
- **全局空间布局**：保持 3:7 非对称分栏，但优化间距与边框，采用更浅的灰底与更柔和的分割线。
- **矩阵导航与微型卡片**：微型卡片需具备轻盈的悬浮阴影、圆润的标签（Pill 形状）以及极简的图标配色。
- **拖拽防呆引擎**：拖拽过程中的合法（绿色）与非法（红色/灰色）反馈需更加柔和，采用高级感半透明色与模糊遮罩。
- **底部智能控制台**：Smart Dock 需具备类似 macOS Dock 或 iOS 灵动岛的悬浮质感，展开动画平滑自然。
